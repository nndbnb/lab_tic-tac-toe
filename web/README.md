# Infinite Tic-Tac-Toe Web Interface

Веб-интерфейс для игры в бесконечные крестики-нолики с AI движком на C++.

## Требования

- Python 3.7+
- Flask (установится автоматически)
- Скомпилированный `web_cli.exe` в папке `../build/`

## Установка

1. Убедитесь, что C++ проект скомпилирован:
   ```bash
   cd ..
   build.bat
   ```

2. Установите зависимости Python:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

Запустите сервер:
```bash
run.bat
```

Или вручную:
```bash
python app.py
```

Откройте браузер и перейдите на `http://localhost:5000`

## Использование

1. **Настройки игры:**
   - Выберите длину для победы (3-20)
   - Выберите, за кого играть (X или O)
   - Выберите, кто ходит первым
   - Установите время на ход AI (100-30000 мс)

2. **Игра:**
   - Кликайте на клетки доски, чтобы сделать ход
   - AI автоматически сделает ход после вашего
   - Статистика поиска отображается в правой панели
   - История ходов показывается внизу

3. **Статистика:**
   - Время поиска
   - Метод решения (Immediate Win, Negamax Search и т.д.)
   - Глубина поиска
   - Количество просмотренных узлов
   - Оценка позиции
   - Principal Variation (последовательность лучших ходов)

## Архитектура

- **Flask сервер** (`app.py`) - обрабатывает HTTP запросы
- **C++ CLI** (`../build/web_cli.exe`) - выполняет игровую логику
- **HTML интерфейс** (`templates/index.html`) - визуализация

Все данные передаются через JSON между компонентами.

## Логирование

Все операции логируются в файл `logs/app.log`:
- Входящие API запросы
- Вызовы C++ CLI с параметрами
- Ответы от C++ CLI
- Состояние игр
- Ошибки и исключения

Логи автоматически ротируются (максимум 10MB на файл, 5 резервных копий).

Для просмотра логов в реальном времени:
```bash
# Windows PowerShell
Get-Content logs\app.log -Wait -Tail 50

# Linux/Mac
tail -f logs/app.log
```
